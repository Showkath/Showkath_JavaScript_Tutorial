JavaScript Basics

This page contains examples of basic JavaScript functions with explanations.

Function: greetUser

This function displays a greeting message to the user.

Function: sumNumbers

This function calculates the sum of two numbers.

Function: generateRandomNumber

This function generates a random number between 1 and 10 and displays it.

Function: checkEvenOdd

This function checks if a number entered by the user is even or odd and displays the result.

Function: displayMultiplicationTable

This function generates and displays the multiplication table of a number entered by the user.

This function capitalizes the first letter of a string entered by the user.

Function: reverseString

This function reverses a string entered by the user.

Function: findLargestNumber

This function finds the largest number from a list of numbers entered by the user.

This function counts the number of vowels in a string entered by the user.

Function: countCharacters

This function counts the number of characters in a string entered by the user.

Function: concatenateStrings

This function concatenates two strings entered by the user.

Function: findLongestWord

This function finds the longest word in a sentence entered by the user.

Function: calculateAverage

This function calculates the average of numbers entered by the user.
