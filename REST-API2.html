Important Concepts

  
1. Promises

  
Promise is a JavaScript object representing the eventual completion or failure of an asynchronous operation, and its resulting value.

  
Example:

  
    const fetchData = new Promise((resolve, reject) => {
      // Simulating async data fetching
      setTimeout(() => {
        const data = { id: 1, name: 'John Doe' };
        // Resolve the Promise with fetched data
        resolve(data);
        // Reject the Promise on failure
        // reject(new Error('Data fetching failed'));
      }, 2000);
    });

    // Using the Promise
    fetchData
      .then(data => console.log('Fetched data:', data))
      .catch(error => console.error('Error:', error));
  


  
2. Asynchronous vs. Synchronous JavaScript

  
Asynchronous JavaScript allows multiple tasks to be executed concurrently, improving performance and responsiveness. Synchronous JavaScript executes tasks one after another, blocking the execution until each task is completed.

  
Example:

  
    // Asynchronous Example (Using setTimeout)
    console.log('Start');
    setTimeout(() => {
      console.log('Async Task Completed');
    }, 1000);
    console.log('End');

    // Synchronous Example (Using for loop)
    console.log('Start');
    for (let i = 0; i < 1000000000; i++) {
      // Heavy computation
    }
    console.log('Sync Task Completed');
    console.log('End');
